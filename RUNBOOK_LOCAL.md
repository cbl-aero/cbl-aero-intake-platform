\# Local Runbook



\## Preconditions

\- Python 3.11

\- Virtualenv active

\- Required packages installed



\## Environment Variables



\### For SharePoint Graph download (optional but recommended)

Set these in bash:

\- MS\_TENANT\_ID

\- MS\_CLIENT\_ID

\- MS\_CLIENT\_SECRET



Example:

```bash

export MS\_TENANT\_ID="your-tenant-id"

export MS\_CLIENT\_ID="your-client-id"

export MS\_CLIENT\_SECRET="your-client-secret"
